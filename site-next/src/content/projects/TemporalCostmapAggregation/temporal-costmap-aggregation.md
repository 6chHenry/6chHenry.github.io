# Temporal Costmap Aggregation

Temporal Costmap Aggregation explores how short-horizon costmap history can make object-relative navigation more robust. The project compares several aggregation strategies over six-frame costmap embeddings and visualizes how they behave under different corruption geometries.

See the project site for the full interactive report, experiment setup, and results.
