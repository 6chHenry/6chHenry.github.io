# 二倍速

我感觉自己活在一个二倍速的世界。不论是抖音，B站还是小红书，我都习惯了长按屏幕开二倍速，有些时候视频还没点击播放，就先把速度设置为二倍速。我连看电影都要二倍速了。

我觉得视频中说话速度都太慢了，但那实际上是正常语速，只是我没有耐心听他们一字一句的说，我只是想了解个大概。甚至连知识类的视频我也习惯了二倍速。

昨天晚上我妹在我旁边，我在长按屏幕，结果她也在我的屏幕上长按，发现速度加快了，她可能也理解了长按的目的，但我很怕把她带坏了。

或许有的视频确实没营养，二倍速可以剩下时间，但是很多时候发现自己错过了关键信息，又要倒回去重新看，不是也会浪费时间吗？最后花费的时间可能和正常速度观看差不太多。

但我应该不会改变这个习惯。
