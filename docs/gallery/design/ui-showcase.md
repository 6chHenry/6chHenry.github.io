## 项目概览

这一年参与的 UI/UX 项目涵盖了多种产品类型——从数据可视化仪表盘到教育类移动应用。

核心关注点：
- 信息架构与用户流程
- 无障碍设计（WCAG 2.1 AA 标准）
- 暗色模式适配
- 响应式布局
