# 河狸变身计划影评

昨天下午看了《河狸变身计划》，我上午还给lzh调侃说我只会去影院看动画片，结果在猫眼一看，上一部一个人去看的是哆啦A梦，在之前是野猪公主，反正基本上都是动画片。我觉得也只有动画片值得我去影院看。

这不电影刚开始20分钟就把我整的眼眶湿润了，真的挺好看的。那种亲情真的很让人感动。

整部电影讲的就是人与自然，很有教育意义吧，但是整个剧情是很套路的，结尾是包饺子的，属于是看到开头就知道结尾的剧情。但是呢，我觉得这不是什么坏事。我们不能以剧情好不好猜作为评判的标准，就算剧情是套路化的，那只是说明历代电影都证明了这样的剧情走向是不会出错暴雷的，那肯定就沿用了啊，这又不是什么科幻片，你有期待有什么反转呢？确实很好看！

我给到9分吧。

小河狸真的很可爱啊，谁懂！
