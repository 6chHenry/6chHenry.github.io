Chapter Four / Production Notes

  1. Season

    冬季还是夏季

    冬季优先 Tromsø、极光和雪地活动；夏季优先峡湾、群岛、徒步与长日照。

  2. Priority

    风景还是城市

    想看风景就固定 Oslo–Bergen 铁路与 Bergen；想看设计与城市，则增加 Stockholm 与 Helsinki。

  3. Buffer

    是否接受转机风险

    Tromsø 需要独立预算和缓冲。五日版本不加入它，是为了保住主线稳定性。

  4. Booking

    先锁哪一段

    先确认国际抵达、Bergen Line 和返程航班，再决定峡湾产品与城市住宿。
